Exercises - Pseudocode

Answer the following questions by implementing the code samples and/or answering the questions in a word document. Upload all project files and word documents zipped to the exercise dropbox.

 

Note: for each of the problems below give well-formed pseudocode, as described in class. An example is given below:

algorithm add(first, second)
//input: integers first and second
//output: the sum of first and second

result <-- first + second
return result

 

1. Give the pseudocode for an algorithm that reorders the elements of the following array:

 A = [x, y, z]  --> A = [y, z, x]

 What is the minimum number of assignment statements needed?

 

2. Give the pseudocode for an algorithm that finds the sum of all integers in an array.

 

3. Give the pseudocode for an algorithm that takes as an input a list of n integers in increasing order and produces a list of all values that occur more than once.

 

4. Give the pseudocode for an algorithm that takes as an input a list of n integers in increasing order and returns the mode of the list (the value that appears most often).

 

5. Give the pseudocode for an algorithm that inserts an integer x in the appropriate position of a list of sorted integers.

Example inputs: A = [2, 5, 13, 21],   x = 10        

Example list after the algorithm completes: A = [2, 5, 10, 13, 21]
